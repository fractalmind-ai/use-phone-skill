# use-phone-skill

Skill instructions for phone/device interaction workflows.

## Compatibility

| Component | Requirement |
|---|---|
| Host OS | Linux/macOS/Windows |
| Runtime | Shell access with script execution permissions |
| Skill File | `SKILL.md` at repo root |
| Scripts | `scripts/` directory present |

## Notes

- Environment-specific permissions may be required for device access.
- Keep local screen coordinate/state files out of shared workflows unless needed.
